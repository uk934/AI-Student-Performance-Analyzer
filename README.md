# 🎓 AI Student Performance Analyzer

## 📌 Overview

This project predicts student final marks based on study hours, attendance, and previous performance using Machine Learning.

## 🚀 Features

* Predict student performance using ML model
* Interactive web app using Streamlit
* Smart feedback based on prediction
* Clean and user-friendly UI

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Streamlit

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Train model:
   python train.py

3. Run app:
   streamlit run app.py

## 📊 Input Features

* Study Hours
* Attendance
* Previous Marks

## 🎯 Output

* Predicted Final Marks
* Performance Feedback
